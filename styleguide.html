Style Guide

This page outlines the design system for the website, including logo, colors, typography, and layout specifications.

Logo

The logo represents the interaction design course, with a modern and bold look. The logo's color is #00C896, which matches the accent color throughout the website.

Logo Color: #00C896 (Teal)

Font: 'D-DIN Bold', sans-serif

Spacing: 10px padding around the logo

Favicon

The favicon matches the primary color of the website to maintain consistency in branding.

Favicon Color: #00C896

Color Palette

The primary colors for the website are designed to create a clean, modern, and professional look. Here are the main colors used throughout the site:

#00C896 - Teal (Accent color)

#4B3F72 - Purple (Primary background color)

#DEDEDE - Light Gray (Secondary background color)

#F4F4F4 - Light background color for sections

Typography

The website uses a combination of bold and clean fonts for readability and a modern look. Here is the breakdown of the fonts used:

Primary Font: Arial, sans-serif

Header Font: 'Arial Black', sans-serif (for h1, bold and uppercase)

Secondary Font: 'D-DIN Condensed', sans-serif (used for subheadings and logo)

Layout

The website layout utilizes a clean, responsive grid system to ensure optimal viewing on various devices. The content is primarily divided into two main sections: header and footer with flexbox for layout control.

Navigation: Flexbox layout with a responsive hamburger menu for smaller screens.

Sections: Main content is divided into sections with flexible widths using flex and box-sizing: border-box for consistency.

Footer: Simple centered footer with background color #4B3F72.
