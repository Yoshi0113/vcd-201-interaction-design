Key Design Decisions

Decision 1: Choosing a Simplified Color Palette

The color palette was carefully selected to ensure consistency and accessibility across all screens. I wanted the design to be modern and clean while maintaining high contrast for readability.

Decision 2: Mobile-first Design

Since a large portion of the user base accesses the site on mobile devices, the design was developed with a mobile-first approach to ensure responsiveness and usability on smaller screens.

Decision 3: User-centered Navigation

After conducting user research, I opted for a sticky navigation bar that allows easy access to different sections of the site. This ensures a smooth user experience as users navigate through the project.
